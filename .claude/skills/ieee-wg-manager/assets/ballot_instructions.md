## Ballot Response Options

You must select ONE of the following responses:

### APPROVE
Select this if you approve the draft as written and have no technical comments.

### DISAPPROVE
Select this if you have technical objections to the draft. **You MUST provide specific technical comments for each disapproval.** Disapprove votes without comments will not be counted.

**Requirements for Disapprove comments:**
- Must be technically substantive
- Must include specific location (section, page, line)
- Must include proposed resolution
- Must be related to the technical content
- Generic objections (e.g., "I don't like it") are not acceptable

### ABSTAIN
Select this if you choose not to vote. Abstentions are not counted in approval percentage but do count toward return rate.

**Valid reasons to abstain:**
- Conflict of interest
- Lack of expertise in specific area
- Insufficient time to review properly

## Frequently Asked Questions

**Q: What if I need more time to review?**
A: Contact the WG chair before the ballot closes to request an extension. Extensions are granted at the discretion of the chair and sponsor.

**Q: Can I change my vote?**
A: Yes, submit a revised ballot before the close date. The latest submission will be counted.

**Q: What if I have editorial comments but approve technically?**
A: Vote Approve and submit editorial comments separately. Editorial comments are always welcome.

**Q: What happens after the ballot closes?**
A: Results are compiled and announced. Disapprove comments are reviewed and resolved. A recirculation ballot may be conducted if significant changes are made.

**Q: How do I know if my comments were addressed?**
A: A comment disposition document will be published showing all comments and their resolutions.

# Concept Explainer

Transform complex academic concepts into clear, memorable understanding.

## Explanation Framework

### 1. The Hook (Why Care?)
Start with why this concept matters:
- Real-world applications
- Connection to student's goals
- Interesting facts or history

### 2. The Core Idea (ELI5)
Explain Like I'm 5 first:
- One sentence summary
- Simple analogy everyone can understand
- No jargon

### 3. Building Blocks
Layer complexity gradually:
- Prerequisites (what you need to know first)
- Key components/parts
- How parts relate to each other

### 4. Concrete Examples
Multiple examples at different levels:
- Simple/familiar example
- Academic example
- Edge case or counterexample

### 5. Common Misconceptions
Address typical confusions:
- What this is NOT
- Frequently confused concepts
- Traps to avoid

### 6. Check Understanding
Interactive verification:
- Ask student to explain back
- Quick diagnostic question
- "If you understood this, you should be able to..."

## Analogy Library

Good analogies connect to:
- Everyday experiences (cooking, sports, games)
- Technology students use (apps, social media)
- Physical/visual intuition

## Depth Levels

Adapt explanation to requested depth:
- **Quick**: 2-3 sentence summary
- **Standard**: Full framework above
- **Deep dive**: Include history, edge cases, connections to other topics

## Example Interaction

Student: "I don't get eigenvalues"

Response:
🎯 **Why eigenvalues matter**: They tell you what happens when you stretch, rotate, or transform something - crucial for everything from Google's search algorithm to quantum physics.

🧠 **The core idea**: An eigenvalue is a special number that tells you how much something stretches (or shrinks) in a particular direction when you apply a transformation.

🎬 **Imagine this**: You have a rubber sheet with arrows drawn on it. When you pull the sheet, most arrows change both direction AND length. But some special arrows only change length - they point in "special directions." The eigenvalue tells you how much those special arrows stretch.

📐 **Formally**: For matrix A, eigenvalue λ, eigenvector v:
Av = λv
"Applying transformation A to v is the same as just scaling v by λ"

❓ **Quick check**: If λ = 2, what happens to the eigenvector when you apply the transformation?
